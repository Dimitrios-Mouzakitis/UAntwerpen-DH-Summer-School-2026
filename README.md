# The Return of Sherlock Holmes: No. 2. The Adventure of the Norwood Builder

This repository contains a TEI/XML digital edition of Arthur Conan Doyle's The Return of Sherlock Holmes, No. 2: The Adventure of the Norwood Builder (holograph manuscript).

The project was developed during the Digital Humanities Summer School 2026 at the University of Antwerp and published as a web edition using GitHub Pages.

## Project Overview

The edition has been encoded according to the Text Encoding Initiative (TEI) Guidelines, preserving the textual structure and relevant manuscript features while enabling digital publication and analysis.

## Technologies

- TEI XML
- HTML
- CSS
- JavaScript
- GitHub Pages
